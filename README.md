# Currency Exchange Office

**Course:** Network Application Development  
**Author:** Kemal Ozer  
**Student ID:** 73799

## Description
A network-based currency exchange office system built with CoreWCF on .NET 8.
Simulates an online exchange office with real exchange rates from the NBP API.

## How to Run

### Start the Service
```bash
cd WCF-Service/CurrencyExchange.Host
dotnet run
```

### Run the Client
```bash
cd WCF-Service/CurrencyExchange.Client
dotnet run
```

## Requirements
- .NET 8 SDK
